[[모던 자바스크립트 Deep Dive]]

소스코드를 4가지로 분류한다.

1. 전역 코드
2. 함수 코드
3. eval 코드
4. 모듈 코드


- 실행 컨텍스트는 소스코드를 실행하는데 필요한 환경을 제공하고 코드의 실행결과를 실제로 관리하는 영역이다.
- 실행 컨텍스트는 식별자를 등록하고 관리하는 스코프와 코드의 실행순서관리를 구현한 내부 매커니즘으로 모든 코드는 실행 컨텍스트를 통해 실행되고 관리된다.
- 식별자와 스코프는 실행 컨텍스트의 렉시컬 환경으로 관리하고 코드 실행순서는 실행 컨텍스트 스택으로 관리한다.
